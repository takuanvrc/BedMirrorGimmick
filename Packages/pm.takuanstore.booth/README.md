# Bed Mirror Gimmick

VRChatのワールド上でベッドミラーが扱えるギミック

## 概要

Bed Mirror Gimmickは、VRChatワールドに簡単にベッドミラー機能を追加できるパッケージです。

## インストール方法

### VCC経由（推奨）

1. VRChat Creator Companion (VCC) を開く
2. Settings → Packages → "Add Repository" をクリック
3. 以下のURLを追加: `https://takuanvrc.github.io/BedMirrorGimmick/index.json`
4. プロジェクトのManage Projectから「Bed Mirror Gimmick」を追加

## 使い方

1. Prefabsフォルダ内のBedMirror Prefabをシーンに配置
2. 位置・回転・スケールを調整
3. 必要に応じてマテリアルをカスタマイズ

## 動作環境

- Unity 2022.3.22f1以降
- VRChat Worlds SDK 3.7.x以降
